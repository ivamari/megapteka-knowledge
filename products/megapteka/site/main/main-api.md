## Страница сайта / (корневая) и /{city}

### MainService
- **Метод:** get
  - **HTTP:** GET
  - **URL:** ma/site/v1/pages/main
  - **Описание:** Данные главной страницы (seo, подборки, баннеры и т.п.). Передаётся city_id автоматически. Доп.параметры в JSON (data).

### mainResolver
- Ожидает загрузку CityService и запрашивает MainService.get().
- Результат прокидывается в компонент через ключ api.

### Роуты
- / — главная (для текущего города по cookie / автоопределению)
- /{city} — главная для конкретного города (city берётся из path, далее city_id).

### Пример запроса
GET ma/site/v1/pages/main?data={"city_id":123}
