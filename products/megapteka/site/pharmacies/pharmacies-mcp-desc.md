# Документация страницы "Подключение аптек"

## Общая информация

- **URL страницы**: https://megapteka.ru/pharmacies
- **Заголовок страницы (title)**: Подключение аптек к сервису Мегаптека - привлечение новых клиентов
- **Description страницы**: Подключите свою аптеку к сервису Мегаптека и получайте новых клиентов. Выгодные условия сотрудничества для аптечных сетей
- **H1 страницы**: Приводим новых клиентов в аптеки
- **Назначение страницы**: Лендинговая страница для привлечения аптек к подключению к сервису Megapteka.ru с презентацией преимуществ сотрудничества
- **Чем полезна страница для пользователя**: Позволяет владельцам и представителям аптек узнать о преимуществах сотрудничества с сервисом и подать заявку на подключение

## Основная область контента

### Навигационные элементы
- **Хлебные крошки**: "Главная" → "Подключение аптек" (ведет на главную страницу /)

### Верхний блок (Hero section)

#### Логотип и награды
- **Логотип сервиса**: Визуальный элемент с брендингом Мегаптека
- **Награды и достижения**: Визуальные элементы с наградами сервиса
- **Ссылка на награду**: "Лучшая система заказа лекарств по всей России" (ведет на соответствующую страницу)

#### Главный заголовок и призыв к действию
- **H1**: "Приводим новых клиентов в аптеки"
- **Подзаголовок**: "Вы продаете СВОЙ ассортимент и по СВОИМ ценам"
- **CTA-кнопка**: "Оставить заявку на подключение" (якорная ссылка, прокручивает к форме заявки)

### Блок преимуществ и статистики

#### Ключевые показатели
- **Посещаемость**: "> 7 млн посещений в месяц"
- **Стоимость привлечения**: "за 30 рублей приводим новых клиентов"
- **Условия продаж**: "Аптеки продают свой ассортимент и по своим ценам"
- **Статистика заказов**: "> 500 000 выкупленных заказов в месяц"

### Дополнительные информационные блоки

#### Блок персонала аптеки
- **Тип**: Компонент `AppPharmacyStaffComponent`
- **Содержание**: Информация о работе с персоналом аптек

#### Быстрая интеграция
- **Тип**: Компонент `FastIntegrationComponent`
- **Содержание**: Информация о процессе интеграции и подключения

### Форма заявки на подключение

#### Поля формы
- **Название компании**: Текстовое поле (обязательное для заполнения)
- **Имя контактного лица**: Текстовое поле для ввода ФИО (обязательное)
- **Email**: Поле с валидацией email-адреса (обязательное)
- **Телефон**: Специальный компонент для ввода номера телефона (необязательное)

#### Элементы управления формой
- **Кнопка "Отправить заявку"**: Отправка данных формы через API
- **Валидация в реальном времени**: Проверка корректности заполнения полей
- **Обработка ошибок**: Отображение сообщений об ошибках валидации

### Функциональные возможности

#### Интерактивные элементы
- **Плавная прокрутка**: При клике на CTA-кнопку страница плавно прокручивается к форме заявки
- **Валидация формы**: Реактивная валидация с проверкой обязательных полей и формата email
- **Отправка заявки**: Интеграция с API для отправки заявок на подключение

#### Аналитика и отслеживание
- **Yandex Metrika**: Отслеживание конверсий и взаимодействий пользователей
- **Метрики эффективности**: Отслеживание успешности подачи заявок
